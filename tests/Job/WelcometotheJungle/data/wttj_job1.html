Le poste

Descriptif du poste

En tant que Développeur Backend PHP / Symfony (H/F), votre rôle consistera à faire évoluer les fonctionnalités existantes (notamment l’amélioration du code, l’écriture de tests, la correction d’erreurs) et à implémenter de nouvelles fonctionnalités.

Intégré(e) à notre pôle Technique (12 personnes), vous rejoindrez l’équipe de François-Xavier (communément nommé FX), Tech Lead sur la partie Backend. L’équipe backend est composée à date de 3 personnes.

Vos missions principales seront les suivantes :

  • Concevoir et développer des fonctionnalités dynamiques et maintenables en PHP Symfony ;

  • Développer des API REST dans un contexte d’échange fluide avec l’équipe Frontend et les équipes Android / iOS ;

  • Structurer le code selon des principes de modularité, clarté, et testabilité ;

  • Participer aux revues de code et à l’implémentation de tests unitaires et fonctionnels ;

  • Travailler en étroite collaboration avec l’équipe Marketing produit, les graphistes et l’équipe Projet pour garantir une expérience utilisateur optimale ;

  • Être force de proposition sur l’amélioration continue des pratiques de développement backend.

Profil recherché

Vous avez au moins 3 ans d’expérience en développement avec le framework Symfony (idéalement 6.4 / 7.3) et PHP (idéalement 8.x).

Une expérience sur une solution de e-commerce (Sylius) et la supervision de l’hébergement de site web est un plus.

Votre formation vous a permis d’acquérir de solides bases en développement web et en programmation orientée objet.

Vous avez une bonne maîtrise de la stack technique suivante : Symfony (principaux composants utilisés : http foundation, console, messenger, mailer, cache, form, validator, http client…), API Platform, PHP, Javascript, HTML/CSS, PHP Storm, Doctrine, Twig, Sylius, Git/Github, Docker.

Des connaissances en infrastructure sont un plus : AWS (EC2, SQS, S3…), Docker, MySQL, Memcached.

Vous êtes attentif(ve) aux tests unitaires et fonctionnels, à la qualité logicielle et à la lisibilité du code. Des compétences en tests automatisés seront un plus.

Vous aimez travailler en équipe tout en sachant être autonome et rigoureux(se).
